# Shop by Category - Image Generation Prompts

## Overview
These prompts are for generating category showcase images for the "Shop by Category" section. Each image should represent the entire category and showcase the variety of products available.

---

## PROMPT 1: Sling Bag Category (15 colors available)

```
Create a premium e-commerce category showcase image for a "Sling Bag" collection featuring handmade jute bags.

TECHNICAL SPECIFICATIONS:
- Aspect Ratio: 4:3 or 16:9 (landscape)
- Resolution: 1920x1440 pixels minimum (or 2560x1440 for 16:9)
- Format: High-quality PNG or JPG
- Color Space: sRGB
- File Size: Optimized but high quality

COMPOSITION & LAYOUT:
- Create a visually appealing arrangement showcasing multiple sling bag styles
- Options for composition:
  * Option A: Three different colored sling bags arranged diagonally (pink, blue, yellow) with one in foreground, two slightly behind
  * Option B: Single prominent sling bag (pink with Mickey design OR blue with butterfly design) as hero, with subtle hints of other bags in background (slightly blurred)
  * Option C: Flat lay style with 2-3 sling bags arranged artistically on a textured surface

PRODUCT REQUIREMENTS:
- Feature rectangular/classic sling bag designs (NOT round)
- Showcase variety: Include bags with different designs like:
  * Mickey/Minnie Mouse face designs
  * Butterfly patterns
  * Checkered patterns
  * Buckle closures
  * Wave patterns
  * Floral designs
- Bags should be in vibrant colors: pink, blue, purple, yellow, black, grey
- Each bag should show unique handcrafted details
- Bags must be in SHARP FOCUS - all details visible
- Show the crossbody strap design clearly

BACKGROUND & SETTING:
- Clean, minimal background that doesn't compete
- Options:
  * Soft beige or cream textured wall
  * Light wooden surface (for flat lay)
  * Warm terracotta or earthy tone
  * Soft gradient background (warm tones)
- Background should be slightly OUT OF FOCUS (bokeh effect) if using multiple products
- Keep it clean, premium, and uncluttered

LIGHTING:
- Natural, soft lighting
- Even illumination across all products
- No harsh shadows
- Products should be well-lit and vibrant
- Warm, inviting light that enhances colors

COLOR PALETTE:
- Vibrant, colorful bags (pink, blue, purple, yellow)
- Warm, earthy background tones
- Overall cheerful, modern aesthetic
- Colors should pop against the background

STYLE & AESTHETIC:
- Premium Indian D2C brand aesthetic
- Modern, youthful, and vibrant
- Handcrafted, artisanal feel
- Sustainable, eco-friendly vibe
- Collection showcase feel (showing variety)

KEY ELEMENTS TO INCLUDE:
- Multiple bag designs visible (to show variety)
- Different colors represented
- Unique features like Mickey faces, butterflies, patterns
- Crossbody strap clearly visible
- Handcrafted texture of jute material

FINAL CHECKLIST:
✓ Shows rectangular/classic sling bag style (not round)
✓ Multiple designs/colors visible (showing variety)
✓ All products in sharp focus
✓ Clean, minimal background
✓ Vibrant, colorful aesthetic
✓ Premium, handcrafted feel
✓ High resolution (1920x1440 minimum)
✓ Suitable for category card display
```
